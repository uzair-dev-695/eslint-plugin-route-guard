# Fastify Basic Example

Basic Fastify application demonstrating route duplicate detection.

## Installation

```bash
npm install
```

## Run ESLint

```bash
npm run lint
```

## What This Demonstrates

- Fastify route detection
- Plugin registration detection
- Duplicate route detection
- Method-specific routes

## Expected Behavior

ESLint should detect the duplicate GET `/users` route in `server.js`.
